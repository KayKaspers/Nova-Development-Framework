# Code of Conduct

## Our Standard

The Nova Development Framework should remain professional, respectful and constructive.

Expected behavior:

- be respectful
- explain decisions clearly
- prefer constructive feedback
- document disagreements
- focus on quality and learning

Unacceptable behavior:

- harassment
- personal attacks
- intentionally destructive contributions
- publishing secrets or private information
- unsafe automation without review

## Enforcement

The Project Owner may reject, revert or block contributions that violate this standard.
