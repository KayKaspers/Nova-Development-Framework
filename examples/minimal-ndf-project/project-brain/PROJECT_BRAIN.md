# Project Brain – Minimal NDF Project

## Projektstatus

Beispielprojekt für NDF Foundation 0.1.

## Ziel

Zeigen, wie ein kleines Projekt nach NDF strukturiert wird.

## Architekturstand

Kein technisches Produkt. Markdown-/YAML-Beispielstruktur.

## Entscheidungen

- Projekt bleibt bewusst klein.
- Kein Backend.
- Kein Docker.
- Fokus auf NDF-Artefakte.
- Level 2 als Startniveau.

## Risiken

- Beispiel darf nicht zu komplex werden.
- Beispiel soll nicht als vollständiges Produkt verstanden werden.
- Inhalte müssen einsteigerfreundlich bleiben.

## Offene Fragen

- Soll später eine kleine HTML-Demo ergänzt werden?
- Soll ein Beispielrelease erstellt werden?
- Soll dieses Beispiel später in der Website erscheinen?

## Lessons Learned
