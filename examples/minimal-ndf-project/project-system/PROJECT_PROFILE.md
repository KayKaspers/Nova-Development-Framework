# Project Profile – Minimal NDF Project

## Kurzbeschreibung

Minimal NDF Project ist ein kleines Beispielprojekt, das zeigt, wie ein Projekt nach dem Nova Development Framework strukturiert wird.

## Problem

NDF enthält viele Konzepte. Ohne Beispiel kann es abstrakt wirken.

## Ziel

Das Projekt soll zeigen, wie NDF-Artefakte praktisch zusammenhängen.

## Zielgruppe

- Kay
- NDF-Nutzer
- Einsteiger
- Projektverantwortliche
- KI-gestützte Entwicklerteams

## Kernfunktionen

- NDF-Projektstruktur zeigen
- Project System demonstrieren
- Work Package Queue demonstrieren
- Rückmeldung an Nova zeigen
- Quality Gates zeigen

## Technische Basis

- Markdown
- YAML
- GitHub
- VS Code
- GitHub Desktop

## Deployment

Kein Deployment erforderlich.

## Aktueller Stand

Basisstruktur vorhanden.

## Bekannte Risiken

- Beispiel könnte zu klein wirken
- Beispiel könnte mit echtem Produkt verwechselt werden
- Inhalte müssen bewusst einfach bleiben

## NDF-Level

Level 2, mit Teilen von Level 3.

## Nächster Meilenstein

Erstes Claude Work Package ausführen und Review dokumentieren.
