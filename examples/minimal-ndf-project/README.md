# Minimal NDF Project

> Ein kleines Beispielprojekt, das zeigt, wie ein Projekt nach dem Nova Development Framework aufgebaut wird.

## Zweck

Dieses Beispiel ist bewusst einfach. Es soll zeigen, wie NDF-Artefakte zusammenarbeiten:

- Project Manifest
- Project Profile
- Project Brain
- Work Packages
- Quality Gates
- Prompts
- Rückmeldung an Nova

## Projektidee

Das Beispielprojekt ist eine kleine Dokumentations-App-Idee:

> Eine einfache lokale Notiz- und Projektübersichtsseite für Lernzwecke.

Es wird kein fertiges Produkt gebaut. Ziel ist der NDF-Ablauf.

## NDF-Level

Start: **Level 2**

Das bedeutet:

- Basisdokumentation vorhanden
- Project Brain vorhanden
- Prompt Workflow vorhanden
- erste Quality Gates vorbereitet

## Empfohlener Workflow

```text
Nova (ChatGPT) plant
Claude setzt kleines Work Package um
Maintainer prüft in VS Code/GitHub Desktop
Maintainer committet und pusht
Nova reviewed
```

## Struktur

```text
minimal-ndf-project/
```

## Erster sinnvoller Schritt

Das erste Work Package erweitert die README und erstellt eine einfache Funktionsbeschreibung.
