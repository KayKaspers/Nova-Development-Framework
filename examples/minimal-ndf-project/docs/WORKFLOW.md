# Workflow – Minimal NDF Project

## Standardablauf

```text
1. Nova definiert Work Package
2. Claude bearbeitet genau dieses Work Package
3. Claude liefert Rückmeldung an Nova
4. Kay prüft Dateien in VS Code/GitHub Desktop
5. Kay committet und pusht
6. Nova reviewed den Stand
```

## Regeln

- keine großen Arbeitspakete
- keine automatischen Git-Operationen durch KI
- keine unklaren Änderungen
- Dokumentation bleibt aktuell
