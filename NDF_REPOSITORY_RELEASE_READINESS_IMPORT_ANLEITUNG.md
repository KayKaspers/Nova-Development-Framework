# Import-Anleitung – NDF Repository Release Readiness v0.1

1. ZIP entpacken.
2. Inhalt in das NDF Repository kopieren.
3. Falls vorhandene Dateien überschrieben werden: Das ist bei README und ähnlichen Dateien beabsichtigt.
4. GitHub Desktop öffnen.
5. Änderungen prüfen.
6. Commit-Nachricht:

```text
chore(repo): add repository release readiness v0.1
```

7. Commit to main.
8. Push origin.

## Hinweis

Die neue `.gitattributes` stabilisiert Zeilenenden zwischen Windows und Linux.
