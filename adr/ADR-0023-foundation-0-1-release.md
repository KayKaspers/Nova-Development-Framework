# ADR-0023: Foundation 0.1 Release

## Status

Accepted

## Context

The Nova Development Framework has reached a state where the main foundation components exist and are documented.

The project now needs a clear release milestone to mark the foundation baseline.

## Decision

NDF Foundation 0.1 is prepared as the first stable foundation release.

The release includes:

- Foundation documentation
- Workflow standards
- Project System
- Prompt Library
- Template Library
- Toolkit
- Academy skeleton
- Export concept
- Website concept
- Branding Bible
- Minimal Example Project

## Consequences

- Future work can target Foundation 0.2 instead of endlessly expanding Foundation 0.1.
- The repository gains a stable milestone.
- Known limitations are documented instead of hidden.
- Real project integration can begin after Foundation 0.1.
