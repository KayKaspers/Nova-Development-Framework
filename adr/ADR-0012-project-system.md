# ADR-0012: NDF Project System

## Status

Accepted

## Kontext

Das NDF benötigt eine strukturierte Methode, um echte Projekte anzubinden.

## Entscheidung

Der einfache Project Adapter wird zu einem Project System erweitert.

Das Project System besteht aus:

- Project Manifest
- Project Profile
- Capability Matrix
- Compliance Check
- Health Score
- Work Package Queue

## Konsequenzen

- Projekte werden messbar.
- Work Packages können systematischer erzeugt werden.
- Projektzustand wird vergleichbar.
- NDF kann später Generatoren und Health Checks unterstützen.
