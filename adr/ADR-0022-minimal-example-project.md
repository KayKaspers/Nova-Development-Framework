# ADR-0022: Minimal NDF Example Project

## Status

Accepted

## Kontext

Das NDF enthält viele Konzepte. Ohne ein vollständiges Beispielprojekt besteht das Risiko, dass Nutzer die praktische Anwendung schwer nachvollziehen können.

## Entscheidung

Ein minimales Beispielprojekt wird unter `examples/minimal-ndf-project/` eingeführt.

## Konsequenzen

- NDF wird praktischer verständlich.
- Project System, Prompt Workflow und Project Brain werden sichtbar.
- Das Beispiel kann später in Academy und Website genutzt werden.
- Größere Projekte können sich daran orientieren, ohne die Struktur 1:1 übernehmen zu müssen.
