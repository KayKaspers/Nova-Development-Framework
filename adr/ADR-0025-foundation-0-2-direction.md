# ADR-0025: Foundation 0.2 Direction

## Status

Accepted

## Kontext

Foundation 0.1 wurde als erster stabiler Foundation-Meilenstein veröffentlicht. Danach soll NDF nicht nur weiter wachsen, sondern praktisch validiert werden.

## Entscheidung

Foundation 0.2 konzentriert sich auf Stabilisierung und Validierung:

- Repository-Konsolidierung
- Basic Automation Checks
- Referenzprojekt B als erste echte Projektintegration
- Academy Band 1 Ausbau
- MkDocs Website-Prototyp
- Export-Prototyp
- Foundation 0.2 Release Criteria

## Konsequenzen

- Neue große Konzepte werden zurückgestellt.
- Der Fokus liegt auf Beweisbarkeit und Anwendung.
- Referenzprojekt B wird zum ersten realen Referenzprojekt.
- Foundation 0.2 wird ein Stabilization Release.
