# ADR-0006: VS Code and GitHub Desktop as Beginner Standard

## Status

Accepted

## Kontext

Cursor erwies sich für Einsteiger als zu komplex.

## Entscheidung

Der empfohlene Standardpfad des NDF ist VS Code + GitHub Desktop. Cursor bleibt optional für Fortgeschrittene.

## Konsequenzen

- niedrigere Einstiegshürde
- weniger Terminal-Probleme
- klarere Verantwortung bei Commits und Pushes
