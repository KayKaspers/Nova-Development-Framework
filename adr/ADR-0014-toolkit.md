# ADR-0014: NDF Toolkit

## Status

Accepted

## Kontext

NDF besitzt inzwischen Foundation, Domain Model, Prompt Library, Template Library und Project System. Diese Bausteine müssen praktisch zusammengeführt werden.

## Entscheidung

Ein NDF Toolkit wird eingeführt.

## Konsequenzen

- Templates, Prompts und Project System werden praktisch nutzbar.
- Ein späterer Generator bekommt eine klare Grundlage.
- Automatisierung wird vorbereitet, aber noch nicht vorschnell umgesetzt.
