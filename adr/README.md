# Architecture Decision Records – Foundation Core

Dieser Ordner enthält die Kern-ADRs des Nova Development Frameworks aus der Foundation-0.1-Phase (ADR-0001 bis ADR-0026).

Foundation 0.1 ist eingefroren; diese ADRs und ihre Nummerierung bleiben unverändert erhalten.

## Abgrenzung

- `adr/` – Foundation-/Framework-Kernentscheidungen (0.1-Ära, eingefroren)
- `docs/adr/` – spätere dokumentationsnahe und thematische ADRs (ab Foundation 0.2, z. B. ADR-0027 ff.)

Neue ADRs entstehen unter `docs/adr/`.

## Bekannte Nummerierungs-Überschneidung

Historisch existieren in `docs/adr/` frühe thematische ADRs mit den Nummern 0001–0005, die sich mit Nummern in diesem Ordner überschneiden, aber teils andere Themen behandeln. Außerdem behandeln `ADR-0002` und `ADR-0006` in diesem Ordner dasselbe Thema (VS Code / GitHub Desktop Standard). Beides ist dokumentierter Altbestand; eine Konsolidierung ist als eigenes Work Package vorgesehen und wird ohne History-Rewrite erfolgen.
