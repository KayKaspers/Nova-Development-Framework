# ADR-0013: NDF Template Library

## Status

Accepted

## Kontext

Das NDF benötigt eine zentrale Vorlagenbibliothek, damit Projekte konsistent gestartet, dokumentiert, geprüft und veröffentlicht werden können.

## Entscheidung

Eine versionierte Template Library wird eingeführt.

## Konsequenzen

- Projekte starten schneller.
- Dokumentation wird einheitlicher.
- Reviews werden strukturierter.
- Releases werden reproduzierbarer.
- Der spätere Generator kann diese Templates nutzen.
