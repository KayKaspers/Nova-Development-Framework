# ADR-0015: Project Starter Manual Flow

## Status

Accepted

## Kontext

Das NDF benötigt eine Möglichkeit, Projekte praktisch zu starten oder anzubinden, bevor ein automatischer Generator existiert.

## Entscheidung

Ein manueller Project Starter Flow wird eingeführt.

## Konsequenzen

- NDF wird sofort praktisch nutzbar.
- Der spätere Generator erhält eine klare fachliche Grundlage.
- Bestehende Projekte können ohne Wartezeit angebunden werden.
