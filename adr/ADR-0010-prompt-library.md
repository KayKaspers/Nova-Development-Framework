# ADR-0010: NDF Prompt Library

## Status

Accepted

## Kontext

Das NDF benötigt konsistente Prompts für KI-gestützte Arbeit. Ohne zentrale Bibliothek entstehen uneinheitliche Arbeitsweisen.

## Entscheidung

Eine versionierte Prompt Library wird eingeführt.

## Konsequenzen

- Prompts werden wiederverwendbar.
- Rückmeldung an Nova wird standardisiert.
- Claude arbeitet in kleineren, prüfbaren Arbeitspaketen.
- Projektadapter können auf Prompts referenzieren.
