# ADR-0002: VS Code and GitHub Desktop as Beginner Standard

## Status
Accepted

## Entscheidung

Der Standard-Workflow für Einsteiger nutzt VS Code als Editor und GitHub Desktop für Git-Operationen.

Cursor bleibt optional für Fortgeschrittene.

## Begründung

Ein Framework für unerfahrene Nutzer darf nicht von komplexen IDE-Agenten abhängen. GitHub Desktop reduziert Fehler und macht Änderungen sichtbar.
