# ADR-0009: Domain Model First

## Status

Accepted

## Kontext

Das NDF wächst zu einem komplexen Engineering Framework. Ohne gemeinsames Fachmodell entstehen inkonsistente Begriffe, Vorlagen und Prompts.

## Entscheidung

Das NDF erhält ein Domain Model als logisches Fundament.

## Konsequenzen

- Prompts, Templates und Academy-Inhalte orientieren sich am Domain Model.
- Neue Begriffe müssen definiert werden.
- Spätere Generatoren und Compliance-Checks können auf dem Modell aufbauen.
