# ADR-0018: NDF Website Concept

## Status

Accepted

## Kontext

Das NDF soll langfristig nicht nur über GitHub nutzbar sein. Einsteiger und Nutzer brauchen eine klare, suchbare und professionell präsentierte Dokumentationsoberfläche.

## Entscheidung

Ein Website Concept wird eingeführt. Die bevorzugte mittelfristige technische Richtung ist MkDocs Material mit GitHub Pages.

## Konsequenzen

- Inhalte bleiben Markdown-basiert.
- Website-Struktur wird früh mitgedacht.
- Academy, Prompt Library, Template Library und Project System werden besser auffindbar.
- Eine spätere Website kann ohne grundlegenden Umbau entstehen.
