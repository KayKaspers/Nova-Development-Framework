# Nova Development Framework

> **Plan. Build. Review. Improve.**

The **Nova Development Framework (NDF)** is a human-led, AI-assisted software engineering framework for structured, documented and quality-focused software projects.

NDF helps turn project ideas into maintainable, reviewable and reusable project systems.

---

## What is NDF?

NDF is a practical framework for:

- planning software projects
- defining architecture and decisions
- creating reusable prompts and templates
- working safely with AI assistants
- documenting project knowledge
- applying quality gates
- preparing releases
- onboarding new projects into a consistent workflow

It is designed for projects that should not grow chaotically, but follow a clear path from idea to release.

---

## Core Principle

> **AI creates. Humans approve.**

AI assistants may help create code, documentation, analysis and reviews.  
Critical decisions, repository history, commits, pushes and releases remain under human control.

---

## Recommended Beginner Workflow

```text
Nova / ChatGPT
  plans, reviews and creates prompts

Claude
  implements small work packages

VS Code
  is used as editor

GitHub Desktop
  is used for commit, pull and push

GitHub
  stores the project history
```

Cursor and advanced automation workflows are optional, not required.

## Key Concepts

### Project System

Defines how real projects are described, evaluated and improved through:

- Project Manifest
- Project Profile
- Capability Matrix
- Compliance Check
- Health Score
- Work Package Queue

### Prompt Library

Reusable prompts for planning, development, documentation, testing, security, reviews and releases.

### Template Library

Reusable project, documentation, architecture, release, testing, security and review templates.

### Project Brain

The long-term memory of a project. It stores decisions, risks, lessons learned, current status and next work packages.

### Work Packages

Small, reviewable units of work that can be safely handed to an AI assistant and checked by a human.
