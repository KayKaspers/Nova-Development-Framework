# Nova Development Framework (NDF)

**One Framework. Every Project.**

Das Nova Development Framework ist ein professioneller Entwicklungsstandard für KI-gestützte Softwareprojekte. Es verbindet Planung, Architektur, Dokumentation, Qualitätssicherung und Umsetzung in einem wiederholbaren Workflow.

## Standard-Workflow für Einsteiger

- **Nova / ChatGPT**: Planung, Architektur, Review, Prompts, Projektsteuerung
- **Claude**: Umsetzung einzelner Arbeitspakete
- **VS Code**: Editor
- **GitHub Desktop**: Commit, Push, Pull ohne Kommandozeile
- **GitHub**: zentrale Projektplattform

Cursor ist optional und gehört zum Fortgeschrittenen-Workflow.

## Projektprinzipien

1. Documentation First
2. Architecture First
3. Human-Governed Git
4. Small Steps
5. Review First
6. Security by Design
7. Modular by Default
8. AI Assisted, Human Approved
