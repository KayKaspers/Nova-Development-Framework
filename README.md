# NDF Prompt Library v0.1

Die NDF Prompt Library ist die zentrale Prompt-Bibliothek des Nova Development Frameworks.

Sie basiert auf dem NDF Domain Model und standardisiert die Zusammenarbeit mit Claude, ChatGPT, Gemini, Copilot und zukünftigen KI-Assistenten.

## Ziel

- wiederverwendbare Prompts
- klare Rollen
- kleine Arbeitspakete
- einheitliche Rückmeldung an Nova
- bessere Qualität
- weniger Kontextverlust
- sicherer Einsatz von KI in Softwareprojekten
