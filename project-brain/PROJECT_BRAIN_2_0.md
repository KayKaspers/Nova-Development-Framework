# Project Brain 2.0

## Zweck

Der Project Brain speichert dauerhaftes Projektwissen.

## Bereiche

- Entscheidungen
- Architektur
- Standards
- Roadmap
- Lessons Learned
- Glossar
- Risiken
- offene Fragen

## Regel

Wenn eine Entscheidung später wichtig sein könnte, gehört sie in den Project Brain.
