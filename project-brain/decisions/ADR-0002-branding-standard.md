# ADR-0002: Branding is a framework standard

## Status

Accepted

## Context

NDF is intended to become the foundation for multiple projects. If every project invents its visual and documentation style from scratch, consistency and recognition suffer.

## Decision

NDF defines a shared branding standard. Projects may adapt it, but they should keep the same basic structure, documentation clarity and accessibility principles.

## Consequences

- New projects start faster.
- Documentation looks more professional.
- Users recognize related projects more easily.
- Branding becomes a reusable project asset instead of a one-time design task.
