# Project Brain – Reference Project First Integration Notes

## Entscheidung

Referenzprojekt B wird als erstes echtes NDF-Referenzprojekt genutzt.

## Ziel

NDF soll beweisen, dass es auf reale, technische Projekte anwendbar ist.

## Wichtig

Die erste Integration ist dokumentarisch. Keine großen Codeänderungen im ersten Schritt.

## Bekannte Referenzprojekt-Learnings

- Preflight 2.0 ist wertvoll als Startgate.
- FFmpeg-Versionen müssen kontrolliert werden.
- Tests sollen externe Binaries mocken.
- Generierte Statusdateien müssen deterministisch sein.
- CI-Gates sollten robust und nachvollziehbar bleiben.
