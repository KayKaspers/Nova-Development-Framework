# Project Brain – Master Plan Notes

## Aktuelle Kernentscheidung

Das NDF wird nicht mehr Cursor-zentriert entwickelt.

Standardpfad:

- ChatGPT/Nova
- Claude bei Bedarf
- VS Code
- GitHub Desktop
- GitHub

Fortgeschrittener Pfad:

- Cursor
- Claude API
- Automatisierung
- Dev Container

## Begründung

Das Framework soll einsteigerfreundlich sein. Cursor bleibt optional, darf aber keine Voraussetzung für erfolgreiches Arbeiten sein.

## Lessons Learned

- GitHub Desktop ist für Einsteiger besser geeignet als Git CLI oder Cursor-Terminal.
- KI-Agenten dürfen keine unverifizierten Git-Operationen ausführen.
- ZIP-Pakete eignen sich für den Start, langfristig soll das Repository die zentrale Wahrheit sein.
