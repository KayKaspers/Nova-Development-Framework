# Lessons Learned

## LL-0001 – Cursor ist optional

Cursor kann mächtig sein, aber für Einsteiger zu komplex. Deshalb ist der Standardpfad VS Code + GitHub Desktop.

## LL-0002 – KI darf Git nicht blind ausführen

Unverifizierte Git-Operationen sind riskant. Commits und Pushes bleiben menschlich kontrolliert.

## LL-0003 – Repository ist die Wahrheit

Der Chat hilft bei Entwicklung, aber dauerhaftes Wissen lebt im Repository.
