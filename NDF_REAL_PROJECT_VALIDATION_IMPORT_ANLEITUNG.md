# Import-Anleitung – NDF Real Project Validation Pack

## Zielrepository

```text
Nova-Development-Framework
```

## Schritte

1. ZIP entpacken.
2. Inhalt in das NDF-Repository kopieren.
3. GitHub Desktop öffnen.
4. Änderungen prüfen.
5. Commit erstellen:

```text
docs(validation): add SpeakCore and CastCore real project validation
```

6. Push origin.
