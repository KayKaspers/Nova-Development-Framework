# Toolkit Commit Checklist

Vor jedem Commit prüfen:

- [ ] Änderungen in GitHub Desktop angesehen
- [ ] keine unerwarteten Dateien
- [ ] keine Secrets
- [ ] Dokumentation angepasst
- [ ] Tests geprüft oder begründet nicht ausgeführt
- [ ] Commit-Nachricht im Conventional-Commits-Stil
- [ ] Rückmeldung an Nova vorhanden, wenn Claude beteiligt war
