# NDF Prompt – Adapt Existing Project to NDF

## Ziel

Binde ein bestehendes Projekt sauber an das Nova Development Framework an.

## Aufgabe

1. Analysiere den aktuellen Projektstand.
2. Erstelle ein Project Manifest.
3. Erstelle ein Project Profile.
4. Erstelle eine Capability Matrix.
5. Erstelle einen Compliance Check.
6. Schätze einen Health Score.
7. Erstelle eine Work Package Queue.
8. Empfiehl die ersten 3 Arbeitspakete.

## Grenzen

- Keine funktionalen Codeänderungen.
- Kein Commit.
- Kein Push.
- Keine großen Umstrukturierungen.

## Rückmeldung an Nova

1. Zusammenfassung
2. Erzeugte Artefakte
3. Risiken
4. Offene Fragen
5. Empfohlene Work Packages
