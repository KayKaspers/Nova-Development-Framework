# NDF Prompt – Architecture Review

## Ziel

Bewerte die Architektur eines Projekts oder Moduls.

## Aufgabe

1. Beschreibe die aktuelle Architektur.
2. Identifiziere Stärken.
3. Identifiziere Schwächen.
4. Prüfe Modularität.
5. Prüfe Wartbarkeit.
6. Prüfe Sicherheitsrisiken.
7. Erstelle Verbesserungsvorschläge.
