# NDF Prompt – Create Project System

## Ziel

Erstelle für ein bestehendes Projekt ein vollständiges NDF Project System.

## Aufgabe

1. Erstelle ein Project Manifest.
2. Erstelle ein Project Profile.
3. Erstelle eine Capability Matrix.
4. Erstelle einen ersten Compliance Check.
5. Erstelle einen initialen Health Score.
6. Erstelle eine Work Package Queue.
7. Nenne Risiken und offene Fragen.

## Grenzen

- Keine Codeänderungen.
- Kein Commit.
- Kein Push.

## Rückmeldung an Nova

Am Ende:

1. Zusammenfassung
2. Erzeugte Dateien
3. Risiken
4. Offene Fragen
5. Empfohlene erste Work Packages
