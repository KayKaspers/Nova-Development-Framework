# Claude Prompt – Dokumentationsaufgabe nach NDF

## Ziel

Erstelle oder verbessere Dokumentation.

## Aufgabe

1. Bestehende Dokumentation prüfen.
2. Fehlende Schritte ergänzen.
3. Sprache einsteigerfreundlich halten.
4. Fachbegriffe erklären.
5. Keine Codeänderungen, außer ausdrücklich gefordert.

## Rückmeldung an Nova

Nutze das Standardformat.
