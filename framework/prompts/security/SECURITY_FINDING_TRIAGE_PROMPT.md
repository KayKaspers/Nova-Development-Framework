## Aufgabe

Für jedes Finding:

1. ID vergeben
2. Titel
3. Severity
4. betroffener Bereich
5. Risiko
6. Exploit-Bedingungen
7. empfohlene Mitigation
8. vorgeschlagener Work-Package-Typ
9. Testbedarf
10. Health-Score-Relevanz
