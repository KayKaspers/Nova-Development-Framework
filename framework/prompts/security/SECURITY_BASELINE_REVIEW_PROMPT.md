# Prompt – Security Baseline Review

## Work Package Type

Security-baseline

## Ziel

Prüfe Security-Basis ohne Codeänderungen.

## Check

- Secrets
- Auth
- Authorization
- Header
- Docker
- Rate Limit
- Audit
- Config Defaults
- CI Security

## Regel

Review-only erhöht Security Score nicht automatisch.

## Rückmeldung an Nova

Findings, Risiken, Empfehlungen, Health Score Hinweis.
