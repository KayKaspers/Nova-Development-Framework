# Claude Prompt – Destructive Action Test Plan

## Rolle

Du bist Claude und erstellst oder prüfst Tests für eine destructive Action.

## Work Package Type

test-only

## Ziel

Ergänze fokussierte Tests für eine bereits freigegebene destructive Action.

## Aufgabe

Teste mindestens:

- valid target
- invalid target
- unmanaged target
- unauthorized actor
- missing/wrong confirmation
- rate limit where possible
- audit privacy
- no unrelated resource affected

## Grenzen

Nur Testdateien ändern, außer Nova erlaubt eine minimale Korrektur. Kein Commit. Kein Push.
