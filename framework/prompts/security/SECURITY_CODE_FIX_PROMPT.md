# Prompt – Focused Security Code Fix

## Work Package Type

Code-fix

## Ziel

Mitigiere genau ein freigegebenes Security Finding.

## Regeln

- scoped files only
- Test ergänzen falls sinnvoll
- keine Secrets loggen
- keine neuen Dependencies ohne Begründung
- kein Commit
- kein Push

## Rückmeldung an Nova

Finding, Dateien, vorher/nachher, Tests, Risiken, Score-Empfehlung.
