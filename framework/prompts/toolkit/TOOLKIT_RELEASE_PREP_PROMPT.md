# NDF Prompt – Release Preparation

## Ziel

Bereite einen Release nach NDF-Standards vor.

## Aufgabe

1. Prüfe README.
2. Prüfe Changelog.
3. Prüfe Release Notes.
4. Prüfe offene Risiken.
5. Prüfe Project Brain.
6. Erstelle Release-Checkliste.
7. Empfehle Go/No-Go.

## Grenzen

- Kein Tag erstellen.
- Kein Release veröffentlichen.
- Kein Push.

## Rückmeldung an Nova

Nutze das Standardformat.
