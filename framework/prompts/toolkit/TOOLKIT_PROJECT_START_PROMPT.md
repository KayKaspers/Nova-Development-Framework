# NDF Prompt – Toolkit Project Start

## Ziel

Bereite ein neues oder bestehendes Projekt für die Arbeit nach NDF vor.

## Aufgabe

1. Stelle die fehlenden Projektinformationen fest.
2. Erstelle Vorschlag für Project Manifest.
3. Erstelle Vorschlag für Project Profile.
4. Erstelle Capability Matrix.
5. Wähle passende Templates.
6. Wähle passende Prompts.
7. Erstelle erste Work Packages.

## Grenzen

- Keine Git-Operationen.
- Keine Codeänderungen.
- Keine Veröffentlichung.

## Rückmeldung an Nova

1. Zusammenfassung
2. Fehlende Informationen
3. Vorgeschlagene Dateien
4. Empfohlene Prompts
5. Erste Work Packages
6. Risiken
