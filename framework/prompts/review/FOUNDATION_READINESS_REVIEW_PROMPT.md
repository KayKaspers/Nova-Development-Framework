# NDF Prompt – Foundation Readiness Review

## Ziel

Prüfe, ob ein NDF Foundation-Meilenstein releasebereit ist.

## Aufgabe

1. Prüfe vorhandene Foundation-Artefakte.
2. Vergleiche sie mit den Release Criteria.
3. Erstelle eine Gap Analysis.
4. Erstelle ein Risk Register.
5. Erstelle nächste Work Packages.
6. Gib Go/No-Go-Empfehlung.

## Grenzen

- Keine Dateien löschen.
- Kein Commit.
- Kein Push.
- Keine Release-Veröffentlichung.

## Rückmeldung an Nova

1. Zusammenfassung
2. Go/No-Go
3. Hauptlücken
4. Risiken
5. Empfohlene nächste Work Packages
