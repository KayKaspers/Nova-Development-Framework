# NDF Prompt – Nova Handoff Review

## Ziel

Erzeuge eine saubere Rückmeldung für Nova nach einem Arbeitsschritt.

## Aufgabe

Fasse zusammen:

1. Was wurde getan?
2. Welche Dateien wurden geändert?
3. Welche Tests wurden ausgeführt?
4. Welche Risiken bleiben?
5. Was sollte Nova als Nächstes prüfen?

## Grenzen

- Keine Änderungen.
- Nur Analyse und Bericht.

## Ausgabe

Nutze ausschließlich das Format Rückmeldung an Nova.
