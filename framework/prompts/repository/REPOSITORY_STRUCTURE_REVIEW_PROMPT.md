# NDF Prompt – Repository Structure Review

## Ziel

Prüfe die Repository-Struktur anhand des NDF Repository Structure Cleanup Plan.

## Aufgabe

1. Prüfe Root-Dateien.
2. Prüfe Ordnerstruktur.
3. Identifiziere doppelte oder temporäre Dateien.
4. Prüfe Prompt- und Template-Standorte.
5. Prüfe Changelog-Dateien.
6. Erstelle empfohlene Cleanup-Work-Packages.

## Grenzen

- Keine Dateien löschen.
- Keine Dateien verschieben.
- Kein Commit.
- Kein Push.

## Rückmeldung an Nova

1. Zusammenfassung
2. Auffälligkeiten
3. Risiken
4. Empfohlene Verschiebungen
5. Empfohlene nächste Work Packages
