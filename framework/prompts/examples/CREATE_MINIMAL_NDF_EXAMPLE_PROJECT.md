# NDF Prompt – Create Minimal Example Project

## Ziel

Erstelle ein kleines Beispielprojekt nach NDF-Standards.

## Aufgabe

1. Erstelle README.
2. Erstelle Roadmap.
3. Erstelle Project Manifest.
4. Erstelle Project Profile.
5. Erstelle Capability Matrix.
6. Erstelle Project Brain.
7. Erstelle Work Package Queue.
8. Erstelle Quality Gates.
9. Erstelle einen Beispiel-Claude-Prompt.
10. Erstelle eine Beispiel-Rückmeldung an Nova.

## Grenzen

- Kein Commit.
- Kein Push.
- Keine produktive Anwendung bauen.
- Beispiel bewusst klein halten.

## Rückmeldung an Nova

1. Zusammenfassung
2. Erzeugte Dateien
3. NDF-Level
4. Risiken
5. Empfohlener nächster Schritt
