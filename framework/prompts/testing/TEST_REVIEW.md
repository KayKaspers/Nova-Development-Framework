# NDF Prompt – Test Review

## Ziel

Bewerte die Teststrategie eines Projekts.

## Aufgabe

1. Prüfe vorhandene Tests.
2. Identifiziere Lücken.
3. Prüfe, ob Tests lokale Systemabhängigkeiten haben.
4. Gib Empfehlungen für stabile Tests.
5. Schlage konkrete Work Packages vor.

## Grenzen

- Keine Änderungen ohne Freigabe.
