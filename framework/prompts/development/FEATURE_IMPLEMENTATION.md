# NDF Prompt – Feature Implementation

## Ziel

Implementiere ein klar definiertes Feature.

## Aufgabe

1. Prüfe vorhandene Struktur.
2. Plane die Änderung kurz.
3. Implementiere das Feature minimal und sauber.
4. Ergänze Tests oder Testhinweise.
5. Ergänze Dokumentation.
6. Liefere Rückmeldung an Nova.

## Grenzen

- Kein Scope Creep.
- Keine Architekturänderung ohne Freigabe.
- Kein Commit.
- Kein Push.

## Rückmeldung an Nova

Nutze den Standardblock Rückmeldung an Nova.
