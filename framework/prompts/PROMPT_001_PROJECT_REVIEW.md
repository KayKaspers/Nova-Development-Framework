# Claude Prompt – Projektreview nach NDF

## Ziel

Analysiere das bestehende Projekt nach NDF-Standards.

## Aufgabe

1. Prüfe Repository-Struktur.
2. Prüfe README und Dokumentation.
3. Prüfe Tests und CI.
4. Prüfe Docker/Deployment, falls vorhanden.
5. Prüfe bekannte Risiken.
6. Erstelle keine großen Änderungen.
7. Liefere eine strukturierte Rückmeldung an Nova.

## Grenzen

- Kein Commit
- Kein Push
- Kein Reset
- Keine automatischen Massenänderungen

## Rückmeldung an Nova

Nutze das Standardformat.
