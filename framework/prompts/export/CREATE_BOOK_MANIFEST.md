# NDF Prompt – Create Book Manifest

## Ziel

Erstelle ein Book Manifest für einen exportierbaren NDF-Band.

## Aufgabe

1. Prüfe vorhandene Kapitel.
2. Lege sinnvolle Reihenfolge fest.
3. Erstelle Manifest nach NDF Book Manifest Specification.
4. Nenne fehlende Kapitel.
5. Nenne Export-Risiken.

## Grenzen

- Keine Dateien löschen.
- Kein Commit.
- Kein Push.
- Keine Build-Ausführung.

## Rückmeldung an Nova

1. Zusammenfassung
2. Erzeugtes Manifest
3. Fehlende Inhalte
4. Risiken
5. Empfohlener nächster Schritt
