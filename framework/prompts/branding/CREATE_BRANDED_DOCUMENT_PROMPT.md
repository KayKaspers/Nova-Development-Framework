# NDF Prompt – Create Branded Document

## Ziel

Erstelle oder überarbeite ein Dokument im Stil der NDF Branding Bible.

## Aufgabe

1. Nutze klare NDF-Sprache.
2. Verwende passende Struktur.
3. Ergänze Zweck, Zielgruppe, Beispiel und nächsten Schritt.
4. Beachte Dokumentstil und Tonalität.
5. Markiere offene Designfragen.

## Grenzen

- Keine Font-Dateien einbinden.
- Kein Commit.
- Kein Push.

## Rückmeldung an Nova

1. Zusammenfassung
2. Geänderte Dateien
3. Designentscheidungen
4. Offene Punkte
5. Empfehlung
