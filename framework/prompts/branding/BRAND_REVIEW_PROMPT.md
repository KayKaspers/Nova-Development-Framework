# NDF Prompt – Brand Review

## Ziel

Prüfe ein NDF-Artefakt anhand der Branding Bible.

## Aufgabe

1. Prüfe Sprache.
2. Prüfe visuelle Konsistenz.
3. Prüfe Lesbarkeit.
4. Prüfe Farb- und Stiltreue.
5. Prüfe Einsteigerfreundlichkeit.
6. Gib konkrete Verbesserungen.

## Grenzen

- Keine Dateien löschen.
- Kein Commit.
- Kein Push.

## Rückmeldung an Nova

1. Zusammenfassung
2. Stärken
3. Abweichungen
4. Verbesserungsvorschläge
5. Go/No-Go
