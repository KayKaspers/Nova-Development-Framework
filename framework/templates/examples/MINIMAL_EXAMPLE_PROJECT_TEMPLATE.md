# Minimal Example Project Template

## Zweck

Diese Vorlage hilft, ein kleines NDF-Beispielprojekt zu erstellen.

## Enthalten sein sollten

- README
- Roadmap
- Project Manifest
- Project Profile
- Capability Matrix
- Project Brain
- Work Package Queue
- Quality Gates
- Beispielprompt
- Rückmeldung an Nova

## Nicht enthalten sein müssen

- produktiver Code
- CI/CD
- Docker
- komplexe Architektur
