# NDF Standard – Owner-only Actions

## Einsatz

Owner-only für irreversible oder besonders riskante Operationen.

## Anforderungen

- serverseitige OWNER-Prüfung
- Warnung
- Read-only Preview
- mehrere Bestätigungen
- Typed Confirmation
- Rate Limit
- Audit
- Tests

## Typed Confirmation

Beispiele:

```text
DELETE BACKUP
DELETE TENANT
RESET PRODUCTION
```
