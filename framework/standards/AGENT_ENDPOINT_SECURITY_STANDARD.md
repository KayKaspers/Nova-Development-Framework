# NDF Standard – Agent Endpoint Security

## Pflichtkontrollen

- Token-Gate
- Scope Validation
- Managed Resource only
- Strict Schema
- Rate Limit
- Audit
- Minimal Response
- Tests

## Verboten

- unauthenticated mutation
- raw command execution
- freie Host-Pfade
- wildcard operations
- secret logging

Mutierende Agent-Endpunkte brauchen ADR.
