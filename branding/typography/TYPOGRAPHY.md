# NDF Typography

## Documentation

Recommended fonts:

- Headings: Inter, Aptos Display, Calibri Light
- Body: Inter, Aptos, Calibri
- Code: JetBrains Mono, Consolas, Cascadia Code

## Rules

- Use short headings.
- Use code formatting for commands, paths and filenames.
- Use tables for comparisons, not for long text.
- Avoid decorative fonts.
- Prioritize readability over style.
