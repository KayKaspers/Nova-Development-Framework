# NDF Color Palette

## Primary Colors

| Name | Hex | Usage |
|---|---:|---|
| Nova Midnight | `#0B1020` | Background, headers, dark UI |
| Framework Blue | `#2563EB` | Primary accent, links, buttons |
| Signal Cyan | `#22D3EE` | Highlights, diagrams, status accents |
| Documentation White | `#F8FAFC` | Page background |
| Slate Text | `#1E293B` | Body text |

## Support Colors

| Name | Hex | Usage |
|---|---:|---|
| Success Green | `#22C55E` | Checks, passed states |
| Warning Amber | `#F59E0B` | Warnings, attention boxes |
| Risk Red | `#EF4444` | Errors, blockers |
| Neutral Gray | `#64748B` | Metadata, borders, secondary text |

## Rule

Use color to guide attention, not as decoration.
