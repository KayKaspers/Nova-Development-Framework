# NDF Project System v0.1

## Zweck

Das NDF Project System beschreibt, wie ein echtes Projekt nach dem Nova Development Framework geführt wird.

Es macht Projekte messbar, vergleichbar und steuerbar.

## Bestandteile

```text
Project System
├── Project Manifest
├── Project Profile
├── Capability Matrix
├── Quality Gates
├── Compliance Check
├── Health Score
├── Work Package Queue
└── Project Brain Link
```

## Warum nicht nur Project Adapter?

Ein Adapter verbindet ein Projekt nur oberflächlich mit dem Framework.

Das Project System geht weiter:

- Es beschreibt die Identität des Projekts.
- Es bewertet Fähigkeiten.
- Es definiert Quality Gates.
- Es erzeugt Arbeitspakete.
- Es unterstützt Reviews.
- Es speichert Lessons Learned.

## Standardablauf

1. Project Manifest erstellen.
2. Project Profile ausfüllen.
3. Capability Matrix bewerten.
4. Quality Gates festlegen.
5. Compliance Check durchführen.
6. Health Score bestimmen.
7. Work Packages ableiten.
8. Project Brain aktualisieren.
