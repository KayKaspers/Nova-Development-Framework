# NDF Core Architecture

## Ebenen

Das NDF besteht aus drei Hauptbereichen:

```text
/docs       Dokumentation und Referenz
/framework  Standards, Templates, Regeln, Prompt-Bibliothek
/academy    Lernmaterial, Übungen, Handbuch
```
