# NDF Project Lifecycle

## Phase 1 – Idea

Projektidee wird beschrieben.

## Phase 2 – Discovery

Ziele, Nutzer, Risiken und technische Optionen werden geprüft.

## Phase 3 – Architecture

Architektur, Module, Datenflüsse und Deployment werden geplant.

## Phase 4 – Foundation Setup

Repository, README, Roadmap, Project Brain und Standards werden erstellt.

## Phase 5 – Implementation

Claude oder Entwickler setzen kleine Arbeitspakete um.

## Phase 6 – Verification

Tests, Reviews, Preflight und Quality Gates prüfen das Ergebnis.

## Phase 7 – Release

Version, Changelog, Release Notes und Dokumentation werden abgeschlossen.

## Phase 8 – Operation

Projekt wird betrieben, gewartet und verbessert.

## Phase 9 – Feedback to NDF

Lessons Learned fließen zurück ins Framework.
