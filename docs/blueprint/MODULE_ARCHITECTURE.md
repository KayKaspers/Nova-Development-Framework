# NDF Module Architecture

## Module

```text
foundation/
academy/
toolkit/
ai/
delivery/
best-practices/
```

## Foundation Module

Regeln, Standards und Governance.

## Academy Module

Didaktische Inhalte.

## Toolkit Module

Nutzbare Vorlagen, Prompts und Generatoren.

## AI Module

KI-Zusammenarbeit und Sicherheitsregeln.

## Delivery Module

Projektabläufe und Release-Prozesse.

## Best Practices Module

Erfahrungen und wiederverwendbares Wissen.

## Modulregel

Jedes Modul braucht:

- Zweck
- Zielgruppe
- Inhalte
- Qualitätskriterien
- Owner
- Roadmap
