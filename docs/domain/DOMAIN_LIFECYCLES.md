# NDF Domain Lifecycles

## Work Package Lifecycle

```text
Draft
→ Approved
→ In Progress
→ Implemented
→ Reviewed
→ Committed
→ Learned
```

## Decision Lifecycle

```text
Proposed
→ Accepted
→ Superseded
→ Deprecated
→ Archived
```

## Standard Lifecycle

```text
Draft
→ Active
→ Under Review
→ Deprecated
→ Replaced
```

## Prompt Lifecycle

```text
Draft
→ Tested
→ Active
→ Improved
→ Deprecated
```

## Project Adapter Lifecycle

```text
Created
→ Applied
→ Reviewed
→ Updated
→ Retired
```

## Lesson Learned Lifecycle

```text
Captured
→ Reviewed
→ Accepted
→ Converted to Standard or Best Practice
```
