# NDF Domain Glossary

## Artifact

Dauerhaftes Projektergebnis.

## Capability

Fähigkeit eines Projekts oder Framework-Moduls.

## Compliance Check

Prüfung auf NDF-Konformität.

## Decision

Dokumentierte Entscheidung.

## Health Score

Bewertung des Projektzustands.

## Knowledge Node

Ein einzelnes Wissenselement im Knowledge Graph.

## Lesson Learned

Erkenntnis aus praktischer Projektarbeit.

## Milestone

Geplanter größerer Fortschritt.

## Module

Logisch abgegrenzter Bestandteil.

## Project

Ein nach NDF geführtes Vorhaben.

## Project Adapter

Bindet ein Projekt an NDF an.

## Project Brain

Langzeitgedächtnis eines Projekts.

## Prompt

Strukturierte KI-Arbeitsanweisung.

## Quality Gate

Verbindlicher Prüfschritt.

## Release

Veröffentlichter Projektstand.

## Review

Strukturierte Prüfung.

## Rule

Konkrete verbindliche Regel.

## Standard

Allgemeine Vorgabe für Arbeitsweise.

## Template

Wiederverwendbare Vorlage.

## Work Package

Kleine umsetzbare Aufgabe.
