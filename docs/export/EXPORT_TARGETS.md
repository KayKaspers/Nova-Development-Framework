# Export Targets

## GitHub Markdown

Direkt im Repository lesbar.

Vorteile:

- sofort verfügbar
- kein Build nötig
- gut für Reviews

## Website

Langfristig empfohlen für öffentliche Dokumentation.

Mögliche Tools:

- MkDocs Material
- Docusaurus
- VitePress

## PDF

Für Handbuch, Druck und Archiv.

Mögliche Tools:

- Pandoc
- Quarto
- Typst
- WeasyPrint

## Word

Für editierbare Dokumente und Weitergabe.

Mögliche Tools:

- Pandoc DOCX
- Python docx
- LibreOffice-Konvertierung

## EPUB

Optional für spätere Buchveröffentlichung.

Mögliche Tools:

- Pandoc
- Calibre
