# Website Structure Concept

## Ziel

Die spätere NDF-Website soll als zentrale Dokumentationsplattform dienen.

## Empfohlene Hauptnavigation

```text
Home
Getting Started
Foundation
Academy
Toolkit
Prompt Library
Template Library
Project System
Best Practices
Reference
```

## Startseite

Die Startseite erklärt:

- Was ist NDF?
- Für wen ist es?
- Wie starte ich?
- Welche Bausteine gibt es?
- Wo finde ich die Academy?
- Wie binde ich ein Projekt an?

## Suchfunktion

Eine Volltextsuche ist langfristig wichtig.

## Dark/Light Mode

Empfohlen über MkDocs Material oder ähnliches Framework.

## Versionierung

Langfristig sollen mehrere Dokumentationsversionen verfügbar sein.
