# Source of Truth

## Grundsatz

Die primäre Quelle des NDF ist das Repository.

## Primärquellen

| Inhalt | Quelle |
|---|---|
| Academy | `academy/` |
| Framework-Dokumentation | `docs/` |
| Standards | `framework/standards/` |
| Prompts | `framework/prompts/` |
| Templates | `framework/templates/` |
| ADRs | `adr/` |
| Project Brain | `project-brain/` |

## Exportierte Dateien

Exportierte Dateien sind abgeleitete Artefakte.

Beispiele:

- PDF
- Word
- Website-Build
- EPUB

## Regel

Exportierte Dateien dürfen nie die einzige Quelle einer Information sein.

Wenn ein PDF geändert werden soll, wird zuerst die Markdown-Quelle geändert.
