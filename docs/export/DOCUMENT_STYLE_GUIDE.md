# Document Style Guide

## Ziel

PDF- und Word-Exporte sollen professionell und konsistent wirken.

## Stilrichtung

- klar
- modern
- technisch
- ruhig
- hochwertig
- einsteigerfreundlich

## Grundlayout

- Titelblatt
- Impressum/Metadaten
- Inhaltsverzeichnis
- Kapitel
- Zusammenfassungen
- Glossar
- Anhänge

## Infoboxen

Typen:

- Hinweis
- Tipp
- Warnung
- Beispiel
- Praxis

## Sprache

- direkte Ansprache
- klare Schritte
- keine unnötig langen Sätze
- Fachbegriffe erklären

## Tabellen

Nur verwenden, wenn sie Inhalte wirklich klarer machen.

## Screenshots

Screenshots benötigen:

- Bildunterschrift
- Kontext
- Erklärung, worauf der Leser achten soll
