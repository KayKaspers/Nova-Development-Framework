# Website Visual Identity

## Ziel

Die Website soll professionell, ruhig und technisch wirken.

## Stil

- modern
- klar
- lesbar
- nicht verspielt
- einsteigerfreundlich
- vertrauenswürdig

## Farben

Finale Farben kommen aus der Branding Bible.

Vorläufige Richtung:

- dunkles Blau/Anthrazit für Technik
- helles Blau/Cyan für Akzente
- neutrale Flächen
- klare Kontraste

## Typografie

Empfohlen:

- Sans-serif für Website
- Monospace für Code
- gute Lesbarkeit auf Desktop und Mobilgeräten

## Diagramme

Diagramme sollen konsistent sein:

- einfache Formen
- klare Beschriftung
- keine überladenen Farben
- lieber verständlich als dekorativ
