# Release Policy

## Versionierung

Das NDF verwendet Semantic Versioning.

## Release-Arten

- Foundation Release
- Minor Release
- Patch Release
- Documentation Release

## Release Checkliste

- README aktuell
- Changelog aktuell
- Roadmap aktualisiert
- ADRs vollständig
- Project Brain aktualisiert
- keine Secrets
- keine ungeklärten losen Dateien
