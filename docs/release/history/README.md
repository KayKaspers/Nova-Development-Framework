# Release History

Dieser Ordner enthält Release-Changelogs und Release-Historie des NDF.

Der zentrale Changelog bleibt:

```text
CHANGELOG.md
```

## Abgrenzung zu docs/import-history/

- `docs/release/history/` – echte Release-Changelogs und Release-Historie (z. B. Foundation-Releases).
- `docs/import-history/` – archivierte Import-Pakete: Import-Anleitungen, Paket-Changelogs, Pack-READMEs.

## Hinweis zu Altbestand

Die hier liegenden Paket-Changelogs aus der Foundation-0.1-Aufbauphase bleiben unverändert erhalten, da Foundation 0.1 eingefroren ist. Neue Paket-Changelogs gehören nach `docs/import-history/`.

## Regel

Neue Release-Historie sollte bevorzugt im zentralen Changelog und in `docs/release/` dokumentiert werden.
