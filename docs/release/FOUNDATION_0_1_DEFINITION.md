# NDF Foundation 0.1 Definition

## Purpose

Foundation 0.1 defines the first stable internal milestone of the Nova Development Framework.

It does not mean the framework is complete. It means the basic operating model is usable.

## Required Components

- Constitution
- Governance
- Master Plan
- Blueprint
- Domain Model
- Practical Workflow
- Prompt Library v0.1
- Project System v0.1
- Project Brain
- ADR System
- Repository Release Readiness

## Acceptance Criteria

Foundation 0.1 is complete when:

- the README explains the project clearly
- core concepts are documented
- AI workflow is defined
- Git safety rules exist
- prompt standards exist
- project adapter/system concept exists
- repository has contribution and security guidance
- ADRs capture major decisions
- Project Brain reflects current strategy

## Not Required for Foundation 0.1

- full Academy
- full generator
- full CLI
- complete website
- all templates
- all project adapters

## Next Phase after Foundation 0.1

Foundation 0.2 should focus on:

- Project System implementation
- Template Library expansion
- Prompt Library expansion
- first real project adapter
- Academy Band 1 skeleton
