# Release Notes – NDF Foundation 0.1

## Release Name

**NDF Foundation 0.1**

## Release Type

Foundation Release

## Summary

NDF Foundation 0.1 establishes the first stable foundation of the Nova Development Framework.

It defines the core principles, structure, workflow, terminology, project system, prompt standards, template system, toolkit concept, academy skeleton, export strategy, website concept, branding foundation and first minimal example project.

This release is not the final version of NDF. It is the first stable base layer.

## Changed

- Cursor is no longer required for the default beginner workflow.
- VS Code and GitHub Desktop are the preferred beginner workflow.
- Project Adapter concept evolved into the broader Project System.
- Repository strategy is currently mono-repo first.

## Known Limitations

- Academy content is mostly skeleton, not yet fully written.
- Website is conceptual, not yet built.
- Export pipeline is conceptual, not yet automated.
- Toolkit is specification-first, not yet a CLI.
- Branding assets are working assets, not final professional identity.
- Automated checks are not yet implemented.
