# Foundation 0.1 Final Release Steps

## 1. Cleanup committen

Commit:

```text
chore(repo): prepare pre-release cleanup
```

## 2. Push origin

In GitHub Desktop:

```text
Push origin
```

## 3. GitHub prüfen

Prüfen:

- README korrekt
- Root sauber
- Release Notes vorhanden
- Minimal Example vorhanden

## 4. GitHub About setzen

Description und Topics gemäß:

```text
docs/repository/GITHUB_ABOUT_SETTINGS.md
```

## 5. Release erstellen

GitHub → Releases → Draft a new release

Tag:

```text
v0.1.0-foundation
```

Title:

```text
NDF Foundation 0.1
```

Beschreibung aus:

```text
docs/release/FOUNDATION_0_1_GITHUB_RELEASE_DRAFT.md
```

## 6. Publish release

Erst veröffentlichen, wenn Kay final geprüft hat.
