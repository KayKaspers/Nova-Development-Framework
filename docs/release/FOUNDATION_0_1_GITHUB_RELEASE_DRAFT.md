```markdown
# NDF Foundation 0.1

This is the first stable foundation milestone of the Nova Development Framework.

NDF Foundation 0.1 establishes the core principles, structure, workflow, Project System, Prompt Library, Template Library, Toolkit concept, Academy skeleton, Export Concept, Website Concept, Branding Bible and Minimal Example Project.

## Highlights

- Human-led, AI-assisted workflow
- Beginner-friendly VS Code + GitHub Desktop workflow
- Project System v0.1
- Prompt Library v0.1
- Template Library v0.1
- Toolkit v0.1
- Academy Band 1 Skeleton
- Export and Website Concepts
- Branding Bible v0.1
- Minimal Example Project

## Known Limitations

- Academy content is still a skeleton.
- Website and export pipeline are not yet implemented.
- Toolkit is specification-first and not yet a CLI.
- Basic automation checks are planned for a later milestone.

## Next

Foundation 0.2 will focus on cleanup, automation, Academy expansion, real project integration and first website/export prototypes.
```
