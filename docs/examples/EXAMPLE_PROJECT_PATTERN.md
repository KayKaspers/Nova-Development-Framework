# Example Project Pattern

## Zweck

Dieses Pattern beschreibt, welche Mindeststruktur ein NDF-Beispielprojekt haben sollte.

## Mindeststruktur

```text
example-project/
├── README.md
├── ROADMAP.md
├── project-system/
│   ├── project-manifest.yaml
│   ├── PROJECT_PROFILE.md
│   ├── CAPABILITY_MATRIX.md
│   ├── COMPLIANCE_CHECK.md
│   ├── HEALTH_SCORE.md
│   └── WORK_PACKAGE_QUEUE.md
├── project-brain/
│   └── PROJECT_BRAIN.md
├── docs/
│   ├── WORKFLOW.md
│   └── QUALITY_GATES.md
└── prompts/
    └── claude/
```

## Regel

Ein Beispielprojekt muss klein, vollständig und verständlich sein.
