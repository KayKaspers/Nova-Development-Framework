# ADR-0003 – GitHub Desktop as Beginner Default

## Status
Accepted

## Entscheidung
Der Einsteiger-Standardworkflow nutzt VS Code und GitHub Desktop. Cursor bleibt optional für Fortgeschrittene.

## Begründung
Das NDF soll für unerfahrene Nutzer verständlich sein. GitHub Desktop reduziert Hürden und macht Änderungen sichtbar.
