# ADR-0001 – Human-Governed Repository Operations

## Status
Accepted

## Entscheidung
KI-Systeme dürfen keine unbestätigten Repository-Operationen durchführen, die Verlauf oder Veröffentlichung verändern.

## Gilt insbesondere für
- commit
- push
- merge
- rebase
- reset
- clean
- force push
- branch delete
- tag delete

## Begründung
Repository-Verlauf ist Projektgeschichte. Veröffentlichungen müssen nachvollziehbar und vom Menschen freigegeben sein.

## Leitsatz
AI creates. Humans approve.
