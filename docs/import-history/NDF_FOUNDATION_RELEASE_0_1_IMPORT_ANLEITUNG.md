# Import-Anleitung – NDF Foundation Release 0.1

## Vorgehen

1. ZIP entpacken.
2. Inhalt in dein GitHub-Repository kopieren.
3. GitHub Desktop öffnen.
4. Änderungen prüfen.
5. Commit-Nachricht verwenden:

```text
feat(foundation): add Foundation Release 0.1 Constitution and Core Architecture
```

6. Commit to main.
7. Push origin.

## Wichtig

Falls Dateien bereits existieren, überschreiben ist gewollt, sofern GitHub Desktop die Änderungen sauber anzeigt.
