# NDF Real Project Validation Pack – Referenzprojekt A + Referenzprojekt B

Dieses Paket überführt reale Projekterkenntnisse aus den Referenzprojekten zurück in das Nova Development Framework.

## Rolle der Referenzprojekte

| Projekt | NDF-Validierung |
|---|---|
| Referenzprojekt A | Security, destructive actions, Backup-Delete, OWNER-only Flows, Audit Privacy, Agent-Endpunkte |
| Referenzprojekt B | Docker, CI, Docs-Stabilität, Security-Baseline, Health Score, lokale Tests, technische Härtung |

## Ziel

NDF wird von einer theoretischen Struktur zu einem praktisch validierten Framework weiterentwickelt.
