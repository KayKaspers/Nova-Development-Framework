# Import-Anleitung – Foundation 0.1 Release Package

1. ZIP entpacken.
2. Inhalt in das Repository `Nova-Development-Framework` kopieren.
3. GitHub Desktop öffnen.
4. Änderungen prüfen.
5. Commit-Nachricht:

```text
chore(release): prepare Foundation 0.1 release package
```

6. Commit to main.
7. Push origin.

## Danach

Nach dem Push kann optional ein GitHub Release vorbereitet werden:

```text
v0.1.0-foundation
```

Das Tagging bitte erst machen, wenn Kay den Stand final geprüft hat.
