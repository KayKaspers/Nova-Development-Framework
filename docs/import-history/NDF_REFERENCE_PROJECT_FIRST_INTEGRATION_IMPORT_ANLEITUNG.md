# Import-Anleitung – Reference Project First Integration Baseline

1. ZIP entpacken.
2. Inhalt in das Repository `Nova-Development-Framework` kopieren.
3. GitHub Desktop öffnen.
4. Änderungen prüfen.
5. Commit-Nachricht:

```text
docs(integrations): add reference project first integration baseline
```

6. Commit to main.
7. Push origin.

## Wichtig

Dieses Paket verändert **nicht** das Referenzprojekt-Repository.

Es dokumentiert im NDF, wie Referenzprojekt B als erstes reales Projekt angebunden wird.
