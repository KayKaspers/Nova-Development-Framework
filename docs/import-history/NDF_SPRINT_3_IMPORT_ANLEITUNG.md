# Importanleitung – Sprint 3

1. ZIP entpacken.
2. Inhalt in das Repository `nova-development-framework` kopieren.
3. GitHub Desktop öffnen.
4. Änderungen prüfen.
5. Commit Summary eintragen:

```text
feat(repo): add Repository Excellence Sprint 3
```

6. Commit to main klicken.
7. Push origin klicken.

Hinweis: Dieser Sprint überschreibt keine bestehenden Inhalte absichtlich. Falls GitHub Desktop Konflikte oder gelöschte Dateien zeigt, vor dem Commit stoppen und Nova fragen.
