# Import-Anleitung – NDF Branding Bible v0.1

1. ZIP entpacken.
2. Inhalt in das Repository `Nova-Development-Framework` kopieren.
3. GitHub Desktop öffnen.
4. Änderungen prüfen.
5. Commit-Nachricht:

```text
docs(branding): add NDF Branding Bible v0.1
```

6. Commit to main.
7. Push origin.

## Hinweis

Dieses Paket enthält SVG-Dateien und CSS/JSON-Farbdefinitionen, aber keine Font-Dateien.
