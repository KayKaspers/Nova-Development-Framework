# Project Adapter Helper

## Zweck

Der Project Adapter Helper führt ein bestehendes Projekt in das NDF.

## Ablauf

1. Projektstatus erfassen.
2. Project Manifest erstellen.
3. Project Profile erstellen.
4. Capability Matrix ausfüllen.
5. Compliance Check durchführen.
6. Health Score schätzen.
7. Erste Work Package Queue erstellen.
8. Passende Prompts auswählen.

## Zielprojekte

- CastCore
- SpeakCore
- AirCore
- SC-OrgaSuite

## Ergebnis

Ein Projekt kann nach NDF weitergeführt werden, ohne neu gestartet werden zu müssen.
