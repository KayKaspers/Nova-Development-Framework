# NDF Toolkit Overview

## Zweck

Das NDF Toolkit stellt praktische Werkzeuge bereit, mit denen Projekte schneller und konsistenter nach NDF aufgebaut werden können.

## Bestandteile

```text
toolkit
├── project-starter
├── prompt-selector
├── template-selector
├── quality-checks
├── release-helper
├── project-adapter-helper
└── future-generator
```

## Was das Toolkit in v0.1 ist

In Version 0.1 ist das Toolkit noch kein fertiges Programm. Es ist eine strukturierte Spezifikation mit Vorlagen, Checklisten und ersten Skript-Konzepten.

## Was das Toolkit später wird

Langfristig kann daraus entstehen:

- CLI Tool
- Projektgenerator
- Health-Checker
- Prompt-Generator
- Template-Generator
- Release-Assistent
- NDF Compliance Checker

## Grundsatz

Das Toolkit automatisiert nur das, was verstanden und dokumentiert ist.
