# Template Selector Concept

## Zweck

Der Template Selector hilft, passende Templates für ein Projekt oder Arbeitspaket auszuwählen.

## Auswahl nach Situation

### Neues Projekt

- Project README Template
- Roadmap Template
- Project Brain Template
- Project Manifest Template
- Capability Matrix Template

### Feature-Arbeit

- Work Package Template
- Test Plan Template
- Documentation Template

### Release

- Release Checklist
- Release Notes
- Changelog

### Review

- Nova Review
- Code Review
- Documentation Review
- Security Review

## Regel

Templates sollen nicht blind kopiert werden. Sie werden passend zum Projekt ausgefüllt und angepasst.
