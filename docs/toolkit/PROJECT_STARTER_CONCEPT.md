# Project Starter Concept

## Ziel

Der Project Starter soll neue Projekte mit NDF-Grundstruktur vorbereiten.

## Eingaben

- Projektname
- Slug
- Zielgruppe
- Projekttyp
- Technologien
- Deployment-Ziel
- Sprache
- Lizenz
- NDF-Level

## Ausgabe

- README
- Roadmap
- Project Brain
- Project Manifest
- Project Profile
- Capability Matrix
- Work Package Queue
- Prompt-Auswahl
- Quality Gates

## Beispiel-Fragen

1. Wie heißt das Projekt?
2. Was ist das Ziel?
3. Wer nutzt es?
4. Ist Docker erforderlich?
5. Wird CI/CD benötigt?
6. Soll das Projekt öffentlich werden?
7. Welche KI-Rolle soll Claude übernehmen?

## NDF-Level

- Level 1: Basisdokumentation
- Level 2: Standards und Project Brain
- Level 3: Quality Gates
- Level 4: Health Score
- Level 5: vollständige NDF-Konformität
