# NDF Level Guide

## Level 0 – Nicht angebunden

Projekt existiert ohne NDF-Struktur.

## Level 1 – Basisanbindung

Erforderlich:

- README
- Roadmap
- Project Manifest
- Project Profile
- erste Work Package Queue

## Level 2 – Standardisiert

Zusätzlich:

- Project Brain
- relevante Standards
- Prompt Workflow
- erste ADRs

## Level 3 – Quality Gates

Zusätzlich:

- Quality Gates
- Compliance Check
- Review Templates
- Test-/Security-Basics

## Level 4 – Health Score

Zusätzlich:

- Health Score
- Capability Matrix
- regelmäßige Reviews
- Lessons Learned

## Level 5 – Vollständig NDF-konform

Zusätzlich:

- Release-Prozess
- vollständige Dokumentation
- Project System vollständig
- kontinuierliche Verbesserung
- Feedback zurück ins NDF

## Empfehlung

Neue Projekte starten mindestens mit Level 2.  
Bestehende Projekte starten zunächst mit Level 1 oder Level 2.
