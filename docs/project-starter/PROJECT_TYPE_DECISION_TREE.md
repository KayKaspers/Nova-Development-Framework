# Project Type Decision Tree

## Frage 1: Ist es ein neues oder bestehendes Projekt?

### Neu

Nutze:

- Project Manifest
- Project Profile
- Roadmap
- Project Brain
- Work Package Queue

### Bestehend

Zusätzlich:

- Compliance Check
- Health Score
- Capability Matrix
- Risikoanalyse

## Frage 2: Hat das Projekt Code?

### Ja

Zusätzlich:

- Test Plan
- CI Review
- Security Review
- Architecture Overview

### Nein

Fokus:

- Dokumentationsstruktur
- Academy/Handbuch
- Release Notes
- Review-Prozess

## Frage 3: Wird es deployed?

### Ja

Zusätzlich:

- Docker Review
- Deployment Guide
- Security Policy
- Release Checklist

## Frage 4: Wird KI genutzt?

### Ja

Zusätzlich:

- Prompt Set
- Rückmeldung-an-Nova-Regel
- Git Safety Standard
- Human Review
