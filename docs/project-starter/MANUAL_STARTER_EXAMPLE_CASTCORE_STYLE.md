# Manual Starter Example – CastCore Style

Dieses Beispiel zeigt, wie ein bestehendes Projekt wie CastCore angebunden werden kann.

## Projekttyp

Bestehende Docker-/Web-/Processing-Suite.

## Empfohlenes NDF-Level

Start: Level 2  
Ziel: Level 4

## Erste Artefakte

- Project Manifest
- Project Profile
- Capability Matrix
- Project Brain
- Compliance Check
- Health Score
- Work Package Queue

## Erste sinnvolle Work Packages

1. Dokumentationsstatus stabilisieren
2. Preflight-Dokumentation prüfen
3. Docker Build-Gates dokumentieren
4. Health Score vorbereiten
5. Release-Readiness prüfen

## Warum dieser Weg?

CastCore besitzt bereits starke technische Grundlagen. NDF sollte daher zuerst Dokumentation, Stabilität und Qualitätsbewertung ergänzen, nicht sofort große neue Features.
