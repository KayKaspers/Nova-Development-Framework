# New Project Flow

## Ziel

Ein neues Projekt sauber nach NDF starten.

## Schritte

### 1. Repository erstellen

Empfohlen:

- GitHub Repository
- README
- Lizenz
- private oder public je nach Ziel

### 2. Project Manifest erstellen

Pfad:

```text
project-system/project-manifest.yaml
```

### 3. Project Profile erstellen

Pfad:

```text
project-system/PROJECT_PROFILE.md
```

### 4. Project Brain erstellen

Pfad:

```text
project-brain/PROJECT_BRAIN.md
```

### 5. Roadmap erstellen

Pfad:

```text
ROADMAP.md
```

### 6. Work Package Queue erstellen

Pfad:

```text
project-system/WORK_PACKAGE_QUEUE.md
```

### 7. Erstes Work Package definieren

Das erste Work Package sollte klein, sicher und prüfbar sein.

### 8. Commit

Commit-Nachricht:

```text
feat(project): initialize NDF project structure
```
