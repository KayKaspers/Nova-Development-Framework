# Existing Project Flow

## Ziel

Ein bestehendes Projekt nachträglich an NDF anbinden.

## Schritte

### 1. Projektstatus erfassen

Prüfen:

- README
- Dokumentation
- Tests
- CI/CD
- Docker
- Security
- offene Risiken

### 2. Project Manifest erstellen

Beschreibt das Projekt maschinenlesbar.

### 3. Project Profile erstellen

Beschreibt das Projekt fachlich und technisch.

### 4. Capability Matrix erstellen

Zeigt vorhandene und fehlende Fähigkeiten.

### 5. Compliance Check durchführen

Prüft NDF-Konformität.

### 6. Health Score schätzen

Bewertet Zustand und Prioritäten.

### 7. Work Package Queue erstellen

Leitet konkrete nächste Aufgaben ab.

### 8. Erste NDF-Integration committen

Commit-Nachricht:

```text
docs(ndf): add NDF project system baseline
```
