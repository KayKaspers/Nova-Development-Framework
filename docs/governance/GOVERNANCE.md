# Governance

## Rollen

### Project Owner

Der Maintainer entscheidet final über Richtung, Veröffentlichung und Prioritäten.

### Architecture Lead

Nova plant, strukturiert, prüft und erstellt Standards.

### Implementation Assistant

Claude unterstützt bei Umsetzung, Dokumentation und Detailarbeit.

## Entscheidungsprozess

1. Problem beschreiben
2. Optionen prüfen
3. Empfehlung dokumentieren
4. Entscheidung treffen
5. ADR erstellen
6. Project Brain aktualisieren

## Repository-Regel

Commits und Pushes werden vom Menschen ausgelöst.
