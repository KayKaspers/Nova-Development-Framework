# Typography Rules

## Grundsatz

NDF nutzt Systemschriften, damit keine Font-Dateien verteilt werden müssen.

## Website

Empfohlen:

```css
font-family: Inter, ui-sans-serif, system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif;
```

## Code

```css
font-family: "JetBrains Mono", "SFMono-Regular", Consolas, "Liberation Mono", monospace;
```

## Dokumente

Für Word/PDF:

- Überschriften: Aptos Display, Segoe UI oder Calibri
- Fließtext: Aptos, Segoe UI oder Calibri
- Code: Consolas

## Größenprinzip

- H1: stark und klar
- H2: Abschnitt
- H3: Unterabschnitt
- Fließtext: gut lesbar
- Code: kleiner, aber nicht zu klein

## Regel

Keine exotischen Fonts als Pflicht. NDF muss auf normalen Systemen sauber lesbar bleiben.
