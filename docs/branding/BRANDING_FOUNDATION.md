# NDF Branding Foundation

## Brand Positioning

The Nova Development Framework (NDF) is a professional, beginner-friendly framework for planning, building, reviewing and improving software projects with human-guided AI assistance.

## Core Message

**Plan. Build. Review. Improve.**

## Brand Personality

- Clear
- Professional
- Calm
- Trustworthy
- Technical, but understandable
- Modular
- Future-ready

## Design Direction

NDF should feel like the foundation layer below all other projects. It should not look playful or overloaded. It should look like a serious software framework with a clean documentation-first identity.

## Visual Principles

1. Clarity before decoration.
2. High contrast and readable typography.
3. Reusable elements for GitHub, documentation, PDF, Word and web.
4. Neutral foundation colors with one strong accent.
5. Diagrams should be understandable without explanation.

## Recommended Slogan

**One Framework. Every Project.**

Secondary slogan:

**Plan. Build. Review. Improve.**
