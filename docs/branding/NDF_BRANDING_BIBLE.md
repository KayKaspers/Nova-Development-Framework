# NDF Branding Bible v0.1

## Zweck

Die Branding Bible definiert, wie das Nova Development Framework visuell, sprachlich und strukturell auftreten soll.

Sie gilt für:

- README
- Website
- Academy
- PDF/Word-Exporte
- Diagramme
- Präsentationen
- GitHub-Grafiken
- Social Preview Cards
- Projektadapter
- spätere NDF-Tools

## Markenidee

NDF steht für geordnete, professionelle und nachvollziehbare Softwareentwicklung mit KI-Unterstützung.

## Kernversprechen

> NDF macht aus Projektideen strukturierte, prüfbare und langfristig wartbare Softwareprojekte.

## Markenprinzipien

### 1. Klarheit

NDF erklärt komplexe Dinge verständlich.

### 2. Struktur

Jede Information hat einen Ort und einen Zweck.

### 3. Verantwortung

KI unterstützt, aber Menschen entscheiden.

### 4. Qualität

Schnelligkeit ist gut. Kontrollierte Qualität ist besser.

### 5. Lernbarkeit

NDF muss auch für Einsteiger verständlich bleiben.

## Leitsatz

```text
Plan. Build. Review. Improve.
```

## Claim

```text
One framework. Every project.
```

## Kernregel

```text
AI creates. Humans approve.
```

## Visuelle Richtung

NDF soll wirken wie:

- professionelles Engineering
- modernes Entwicklerwerkzeug
- klare Dokumentationsplattform
- ruhige technische Marke
- nicht verspielt
- nicht überladen
- nicht aggressiv

## Sprachliche Richtung

NDF spricht:

- klar
- direkt
- ruhig
- hilfreich
- strukturiert
- einsteigerfreundlich
- nicht überheblich
- nicht unnötig akademisch
