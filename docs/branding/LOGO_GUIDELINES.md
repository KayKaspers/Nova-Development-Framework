# Logo Guidelines

## Logo-Idee

Das NDF-Logo basiert auf drei Elementen:

1. Nova-Stern / Orientierung
2. Framework-Gitter / Struktur
3. Monogramm NDF / Wiedererkennbarkeit

## Primärlogo

Verwendung:

- Website Header
- README
- Dokumenttitel
- Release-Grafiken

## Markenzeichen

Kompakte Version ohne Text.

Verwendung:

- Favicon
- GitHub Avatar
- kleine Badges
- Icons

## Schutzraum

Um das Logo soll immer ausreichend Abstand bleiben.

## Nicht erlaubt

- verzerren
- schlechte Kontraste
- zu viele Effekte
- harte Schatten
- fremde Farben ohne Zweck
- Logo mit Text überladen

## Varianten

- Dark Mode Logo
- Light Mode Logo
- Monochrome Logo
- Badge-Version

## Hinweis

Die SVG-Dateien in diesem Paket sind v0.1-Arbeitsassets. Sie können später durch ein finales professionelles Logo ersetzt werden.
