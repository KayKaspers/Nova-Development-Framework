# Website Style

## Ziel

Die NDF-Website soll wie eine professionelle Entwicklerdokumentation wirken.

## Eigenschaften

- schnelle Orientierung
- klare Navigation
- gute Suche
- Dark/Light Mode möglich
- responsive
- strukturierte Startseite
- klare Call-to-Actions

## Startseiten-Stil

Hero:

```text
Nova Development Framework
Plan. Build. Review. Improve.
```

Kurzbeschreibung:

> A human-led, AI-assisted framework for structured software projects.

## Karten

Karten können genutzt werden für:

- Foundation
- Academy
- Toolkit
- Prompt Library
- Template Library
- Project System

## Buttons

- Getting Started
- Use Project Starter
- Explore Prompt Library
- Read Academy

## Regel

Die Website soll nicht wie Werbung wirken, sondern wie eine hochwertige Dokumentationsplattform.
