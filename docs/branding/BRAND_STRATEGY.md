# NDF Brand Strategy

## Positionierung

NDF ist ein Software Engineering Framework für Menschen, die Projekte mit KI-Unterstützung professionell planen, bauen und pflegen wollen.

## Zielgruppen

### Einsteiger

Brauchen klare Schritte, Erklärungen und Sicherheit.

### Projektverantwortliche

Brauchen Struktur, Standards und Nachvollziehbarkeit.

### Entwickler

Brauchen Templates, Prompts, Workflows und Qualitätsregeln.

### KI-gestützte Projektteams

Brauchen klare Rollen, Grenzen und Feedback-Schleifen.

## Markenarchitektur

```text
Nova Development Framework
├── Foundation
├── Academy
├── Toolkit
├── Prompt Library
├── Template Library
├── Project System
└── Best Practices
```

## Markenwerte

- Klarheit
- Verantwortung
- Qualität
- Wiederverwendbarkeit
- Lernbarkeit
- Stabilität

## Nicht-Ziele

NDF soll nicht wirken wie:

- ein Hype-Projekt
- ein reines KI-Spielzeug
- ein unübersichtlicher Prompt-Sammelordner
- eine starre Konzernmethode
- ein rein theoretisches Modell

## Tonalität

Praktisch, ruhig, präzise und motivierend.
