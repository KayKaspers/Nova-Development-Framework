# NDF Color System

## Primärfarben

| Name | Hex | Zweck |
|---|---|---|
| NDF Deep Navy | `#0B1220` | Haupt-Hintergrund, technische Tiefe |
| NDF Slate | `#172033` | Flächen, Karten, Navigation |
| NDF Cyan | `#38BDF8` | Primärer Akzent |
| NDF Blue | `#2563EB` | Links, Buttons, Highlights |
| NDF Ice | `#E0F2FE` | helle Akzentflächen |

## Sekundärfarben

| Name | Hex | Zweck |
|---|---|---|
| NDF Mint | `#2DD4BF` | positive Hinweise |
| NDF Amber | `#F59E0B` | Warnungen |
| NDF Red | `#EF4444` | Fehler/Risiken |
| NDF Gray | `#94A3B8` | Sekundärtext |
| NDF White | `#F8FAFC` | Text auf dunklem Hintergrund |

## Farbrollen

### Hintergrund

- Dunkel: `#0B1220`
- Karten: `#172033`
- Helle Dokumente: `#FFFFFF`

### Text

- Dunkelmodus: `#F8FAFC`
- Sekundär: `#94A3B8`
- Hellmodus: `#0F172A`

### Akzent

- Hauptakzent: `#38BDF8`
- Link/Action: `#2563EB`

## Regel

Farbe dient der Orientierung, nicht der Dekoration.

## Barrierefreiheit

- Ausreichender Kontrast ist Pflicht.
- Warnungen dürfen nicht nur über Farbe erkennbar sein.
- Icons oder Labels ergänzen Farbcodes.
