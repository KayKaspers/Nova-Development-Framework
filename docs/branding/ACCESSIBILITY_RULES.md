# Accessibility Rules

## Ziel

NDF soll gut lesbar und zugänglich sein.

## Regeln

- ausreichender Farbkontrast
- klare Linktexte
- keine Information nur über Farbe
- Bilder mit Alt-Text
- Tabellen mit klaren Überschriften
- kurze Absätze
- verständliche Sprache
- Tastaturbedienung für Website berücksichtigen

## Diagramme

Diagramme brauchen eine textliche Erklärung.

## Icons

Icons sind Ergänzung, nicht alleiniger Informationsträger.

## Dokumente

PDF/Word-Exporte sollen:

- klare Überschriftenstruktur haben
- Inhaltsverzeichnis enthalten
- Bilder beschreiben
- keine winzigen Schriftgrößen verwenden
