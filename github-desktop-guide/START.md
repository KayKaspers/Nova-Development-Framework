# GitHub Desktop Schnellstart

## Ziel
Dieses Dokument erklärt den einfachsten Weg, ein NDF-Projekt ohne Kommandozeile zu verwalten.

## Repository klonen

1. GitHub Desktop öffnen.
2. File → Clone repository wählen.
3. Repository auswählen.
4. Lokalen Speicherort festlegen.
5. Clone klicken.

## Änderungen übernehmen

1. Dateien in VS Code bearbeiten.
2. GitHub Desktop öffnen.
3. Änderungen links prüfen.
4. Commit-Nachricht eingeben.
5. Commit to main klicken.
6. Push origin klicken.

## Sicherheitsregel
Wenn du unsicher bist: nicht committen, sondern Nova fragen.
