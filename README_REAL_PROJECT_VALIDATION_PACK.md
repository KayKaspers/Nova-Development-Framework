# NDF Real Project Validation Pack – SpeakCore + CastCore

Dieses Paket überführt reale Projekterkenntnisse aus SpeakCore und CastCore zurück in das Nova Development Framework.

## Rolle der Referenzprojekte

| Projekt | NDF-Validierung |
|---|---|
| SpeakCore | Security, destructive actions, Backup-Delete, OWNER-only Flows, Audit Privacy, Agent-Endpunkte |
| CastCore | Docker, CI, Docs-Stabilität, Security-Baseline, Health Score, lokale Tests, technische Härtung |

## Ziel

NDF wird von einer theoretischen Struktur zu einem praktisch validierten Framework weiterentwickelt.
