# Import-Anleitung – NDF Master Plan

## Schritte

1. ZIP entpacken.
2. Inhalt in das Repository `Nova-Development-Framework` kopieren.
3. In GitHub Desktop prüfen, welche Dateien geändert wurden.
4. Commit-Nachricht verwenden:

```text
docs(master-plan): add NDF Master Plan
```

5. Commit to main.
6. Push origin.

## Nach dem Push

Nova informieren mit:

```text
Master Plan ist gepusht.
```
