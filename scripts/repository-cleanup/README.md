# Repository Cleanup Scripts

Dieser Ordner enthält Hilfsskripte für kontrollierte Repository-Aufräumarbeiten.

## Skript

```text
cleanup-root-imports.ps1
```

## Zweck

Verschiebt historische Import-Anleitungen und Paket-Changelogs aus dem Repository-Root in passende Archivordner.

## Sicherheitsprinzip

Standardmäßig wird nur angezeigt, was passieren würde.

Zum Anwenden muss `-Apply` gesetzt werden.
