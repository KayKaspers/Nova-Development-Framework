# NDF Branding Sprint 2 Import Anleitung

## Ziel

Dieses Paket ergänzt das NDF um die Branding Foundation.

## Schritte

1. ZIP entpacken.
2. Inhalt in das Repository `Nova-Development-Framework` kopieren.
3. GitHub Desktop öffnen.
4. Änderungen prüfen.
5. Commit Summary eintragen:

```text
feat(branding): add Branding Foundation Sprint 2
```

6. Commit klicken.
7. Push origin klicken.

## Hinweis zu Zeilenenden

Dieses Paket enthält eine `.gitattributes` Datei. Sie reduziert künftig Warnungen zu LF/CRLF-Zeilenenden.
