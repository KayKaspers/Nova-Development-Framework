# Workflow Standard

## Rollen

### Kay / Mensch
Projektleitung, Freigabe, Tests, Commit und Push.

### Nova / ChatGPT
Planung, Architektur, Dokumentation, Reviews, Prompts.

### Claude
Umsetzung klarer Arbeitspakete.

### VS Code
Editor für Dateien.

### GitHub Desktop
Git-Verwaltung ohne Kommandozeile.

## Standardablauf

1. Nova plant.
2. Claude setzt ein Arbeitspaket um.
3. Mensch prüft in VS Code.
4. Mensch prüft Änderungen in GitHub Desktop.
5. Mensch committet und pusht.
6. Nova reviewt den Stand.
