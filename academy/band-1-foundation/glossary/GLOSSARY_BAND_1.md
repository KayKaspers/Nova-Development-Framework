# Glossar – Band 1

## ADR

Architecture Decision Record. Eine dokumentierte Architekturentscheidung.

## Branch

Ein Entwicklungszweig in Git.

## Claude

KI-Assistent für Umsetzung, Dokumentation und Analyse.

## Commit

Ein gespeicherter Änderungsstand.

## Git

System zur Versionsverwaltung.

## GitHub

Plattform zum Speichern und Verwalten von Repositories.

## GitHub Desktop

Einfache Oberfläche für Git.

## NDF

Nova Development Framework.

## Nova

Architektur-, Review- und Planungsrolle im NDF.

## Project Brain

Langzeitgedächtnis eines Projekts.

## Pull

Änderungen von GitHub auf den lokalen PC holen.

## Push

Lokale Änderungen zu GitHub übertragen.

## README

Startdokument eines Projekts.

## Repository

Projektordner mit Versionshistorie.

## VS Code

Editor zum Bearbeiten von Projektdateien.

## Work Package

Kleine, klar beschriebene Aufgabe.
