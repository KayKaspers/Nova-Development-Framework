# NDF Academy – Band 1

## Titel

**Vom absoluten Anfänger zum ersten professionellen Projekt**

## Zielgruppe

Dieser Band richtet sich an Menschen ohne tiefes IT- oder Entwicklerwissen.

## Lernziel

Am Ende von Band 1 kann der Leser:

- ein GitHub-Repository verstehen
- VS Code und GitHub Desktop nutzen
- mit Nova und Claude nach NDF arbeiten
- ein Projekt nach NDF starten
- Änderungen prüfen, committen und pushen
- einfache Quality Gates anwenden
