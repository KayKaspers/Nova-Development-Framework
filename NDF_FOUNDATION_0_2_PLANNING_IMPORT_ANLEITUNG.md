# Import-Anleitung – NDF Foundation 0.2 Planning Package

1. ZIP entpacken.
2. Inhalt in das Repository `Nova-Development-Framework` kopieren.
3. GitHub Desktop öffnen.
4. Änderungen prüfen.
5. Commit-Nachricht:

```text
docs(roadmap): add Foundation 0.2 planning package
```

6. Commit to main.
7. Push origin.

## Hinweis

Dieses Paket startet Foundation 0.2.  
Es verändert keine Foundation-0.1-Release-Tags.
